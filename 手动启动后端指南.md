# 🔧 手动启动后端服务指南

## 步骤 1: 激活虚拟环境

在终端中执行以下命令激活虚拟环境：

### Windows (Git Bash)

```bash
source venv/bin/activate
```

### Windows (PowerShell)

```powershell
.\venv\Scripts\Activate.ps1
```

### Windows (CMD)

```cmd
venv\Scripts\activate.bat
```

激活成功后，命令提示符前会显示 `(venv)`。

## 步骤 2: 进入后端目录

```bash
cd backend
```

## 步骤 3: 启动后端服务

```bash
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

或者使用：

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

## 步骤 4: 验证服务

看到以下信息表示启动成功：

```
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
INFO:     Application startup complete.
```

然后可以在浏览器中访问：
- **健康检查**: http://localhost:8000/api/health
- **API 文档**: http://localhost:8000/docs

## 完整启动命令（一行）

```bash
source venv/bin/activate && cd backend && python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

## 停止服务

在终端中按 `Ctrl+C` 即可停止服务。

## 常见问题

### 1. 找不到 Python 模块

如果提示找不到 uvicorn 或 fastapi，请先安装依赖：

```bash
pip install -r backend/requirements.txt
```

### 2. 端口被占用

如果 8000 端口被占用，可以更换端口：

```bash
python -m uvicorn main:app --host 0.0.0.0 --port 8001 --reload
```

### 3. 数据库连接错误

后端服务可以正常启动，但某些数据库相关功能可能不可用。这不影响基本的 API 功能。
