# 快速开始指南

## 🚀 立即验证新功能

### 1. 测试消息生成功能

```bash
# 运行消息生成演示
python3 demo_message_generation.py
```

这将演示：
- 默认值生成
- 随机值生成
- 边界值生成
- 异常值生成
- 异常注入
- 二进制编码
- 完整工作流程

### 2. 测试RAG性能

```bash
# 运行RAG性能测试
python3 test_rag_performance.py
```

这将验证：
- 平均检索延迟是否≤1500ms
- Top-3命中率是否≥90%

### 3. 启动API服务器

```bash
# 启动后端服务
python3 backend/main.py
```

然后访问：
- API文档：http://localhost:8000/docs
- 健康检查：http://localhost:8000/api/health

### 4. 测试新API接口

#### 4.1 生成消息实例

```bash
curl -X POST "http://localhost:8000/api/message-generation/generate-instance" \
  -H "Content-Type: application/json" \
  -d '{
    "message_definition": {
      "label": "J3.2",
      "title": "Air Track Message",
      "version": "6016B",
      "fields": [
        {
          "name": "Track Number",
          "dtype": "uint16",
          "bits": [0, 15],
          "constraint": {
            "required": true,
            "min_value": 0,
            "max_value": 65535
          }
        }
      ]
    },
    "mode": "random",
    "fill_header": true,
    "count": 3
  }'
```

#### 4.2 注入异常

```bash
curl -X POST "http://localhost:8000/api/message-generation/inject-anomaly" \
  -H "Content-Type: application/json" \
  -d '{
    "message_definition": {...},
    "instance": {
      "Track Number": 12345,
      "message_type": "J3.2"
    },
    "strategy": "type_mismatch",
    "count": 1
  }'
```

#### 4.3 编码为二进制

```bash
curl -X POST "http://localhost:8000/api/message-generation/encode-binary" \
  -H "Content-Type: application/json" \
  -d '{
    "message_definition": {...},
    "instance": {...},
    "format_type": "raw",
    "endianness": "big",
    "alignment": 4
  }'
```
