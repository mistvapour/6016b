# 🚀 快速启动后端服务

## 问题：前端无法连接后端

错误信息：`ERR_CONNECTION_REFUSED` 表示后端服务未运行。

## ✅ 解决方法：启动后端服务

### 方法 1: 在终端中启动（推荐）

在 Git Bash 中运行：

```bash
cd backend
source .venv/Scripts/activate
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

### 方法 2: 使用批处理文件

双击运行 `backend/启动后端.bat`

## 📋 启动成功标志

看到以下输出表示启动成功：

```
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
INFO:     Started reloader process
INFO:     Started server process
INFO:     Waiting for application startup.
INFO:     Application startup complete.
```

## 🔍 验证后端服务

启动后，在浏览器访问：
- http://localhost:8000/api/health

如果返回 JSON 数据，说明后端正常运行。

## 🔗 前后端连接

- **前端地址**: http://localhost:5173
- **后端地址**: http://localhost:8000
- **前端会自动通过代理连接后端**

## ⚠️ 重要提示

1. **必须同时运行前后端服务**：
   - 后端：`python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload`
   - 前端：`npm run dev`（在 frontend 目录）

2. **后端服务必须在前端之前启动**，否则前端无法连接

3. **不要直接运行 `python main.py`**，会导致 Segmentation fault

## 🛑 停止服务

按 `Ctrl+C` 即可停止后端服务
