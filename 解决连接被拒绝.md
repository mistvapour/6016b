# ❌ 连接被拒绝错误 - 快速解决

## 错误信息

```
ERR_CONNECTION_REFUSED
无法访问此网站
127.0.0.1 拒绝了我们的连接请求
```

## ✅ 原因

后端服务没有运行！

## 🚀 解决方法

### 方法 1: 双击启动脚本（最简单）

双击运行以下文件之一：
- **`backend/启动后端.bat`** - 只启动后端
- **`启动后端.bat`** - 启动后端（在项目根目录）
- **`start_services.bat`** - 同时启动前后端

### 方法 2: 在终端中启动

**打开一个新的终端窗口**（Git Bash、PowerShell 或 CMD），运行：

```bash
cd backend
source .venv/Scripts/activate
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

或者在 PowerShell/CMD 中：

```cmd
cd backend
.venv\Scripts\activate.bat
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

## 📋 启动成功的标志

看到以下输出表示后端已启动：

```
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
INFO:     Started reloader process
INFO:     Started server process
INFO:     Application startup complete.
```

## 🔍 验证

启动后端后，刷新浏览器页面或访问：
- http://127.0.0.1:8000/api/health
- http://localhost:8000/docs

如果能看到响应，说明后端正常运行！

## ⚠️ 重要提示

1. **后端必须在独立终端窗口中运行**，不要关闭那个窗口
2. **前端和后端需要同时运行**才能正常工作
3. **使用 `Ctrl+C` 停止后端服务**

## 🎯 快速启动步骤

1. 双击运行 `backend/启动后端.bat`
2. 等待看到 "Uvicorn running" 消息
3. 刷新浏览器页面

完成！
