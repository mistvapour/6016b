# 项目依赖安装指南

## ⚠️ 重要提示

检测到您的系统缺少以下必要环境：
1. ❌ Xcode Command Line Tools（macOS必需）
2. ❌ Node.js（前端需要）
3. ❌ Python pip（后端需要）

## 📋 安装步骤

### 步骤 1: 安装 Xcode Command Line Tools

这是macOS上所有开发工具的基础，必须首先安装。

#### 方法一：自动安装（推荐）
```bash
# 在终端中运行以下命令
xcode-select --install
```

这将会弹出安装对话框，点击"安装"即可。安装过程需要10-30分钟，取决于网速。

#### 方法二：手动下载
如果自动安装失败，可以访问：
https://developer.apple.com/download/all/
搜索并下载 "Command Line Tools for Xcode"

---

### 步骤 2: 安装 Homebrew（推荐）

Homebrew 是 macOS 上最好的包管理器，建议安装。

```bash
# 安装 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装后添加到PATH（如果提示）
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
source ~/.zshrc
```

---

### 步骤 3: 安装 Python 环境

#### 使用 Homebrew 安装 Python
```bash
# 安装最新版 Python
brew install python@3.11

# 验证安装
python3 --version
pip3 --version
```

#### 或者使用官方安装包
访问 https://www.python.org/downloads/ 下载安装

---

### 步骤 4: 安装 Node.js

#### 使用 Homebrew 安装 Node.js
```bash
# 安装 Node.js（包含 npm）
brew install node

# 验证安装
node --version
npm --version
```

#### 或者使用官方安装包
访问 https://nodejs.org/ 下载 LTS 版本

---

### 步骤 5: 安装后端依赖

安装 Xcode Command Line Tools 和 Python 后：

```bash
# 进入项目目录
cd /Users/feilubi/Documents/GitHub/6016b

# 安装后端依赖
python3 -m pip install --upgrade pip
python3 -m pip install -r backend/requirements.txt

# 如果某些包安装失败，可能需要安装系统依赖
# OpenCV 需要
brew install opencv

# Camelot 需要
brew install ghostscript
brew install imagemagick
```

---

### 步骤 6: 安装前端依赖

```bash
# 进入前端目录
cd frontend

# 安装前端依赖
npm install

# 如果需要清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

---

### 步骤 7: 安装论文模板依赖（可选）

论文模板使用 LaTeX，如果需要编译 PDF：

```bash
# 安装 macOS TeX 发行版（MacTeX）
brew install --cask mactex

# 或者轻量版本（推荐）
brew install --cask basictex

# 安装后添加命令到PATH
eval "$(/usr/libexec/path_helper)"

# 安装 PlantUML 支持
brew install graphviz

# Python 依赖
pip3 install matplotlib plantuml requests
```

## 🔧 验证安装

创建测试脚本验证所有依赖：

```bash
# 运行验证脚本
cat > check_dependencies.sh << 'EOF'
#!/bin/bash

echo "=== 检查系统依赖 ==="

# 检查 Xcode Command Line Tools
if xcode-select -p &>/dev/null; then
    echo "✓ Xcode Command Line Tools 已安装"
else
    echo "✗ Xcode Command Line Tools 未安装 - 运行: xcode-select --install"
fi

# 检查 Python
if command -v python3 &> /dev/null; then
    PYTHON_VERSION=$(python3 --version 2>&1)
    echo "✓ Python: $PYTHON_VERSION"
else
    echo "✗ Python 未安装"
fi

# 检查 pip
if command -v pip3 &> /dev/null; then
    PIP_VERSION=$(pip3 --version 2>&1)
    echo "✓ pip: $PIP_VERSION"
else
    echo "✗ pip 未安装"
fi

# 检查 Node.js
if command -v node &> /dev/null; then
    NODE_VERSION=$(node --version 2>&1)
    echo "✓ Node.js: $NODE_VERSION"
else
    echo "✗ Node.js 未安装"
fi

# 检查 npm
if command -v npm &> /dev/null; then
    NPM_VERSION=$(npm --version 2>&1)
    echo "✓ npm: $NPM_VERSION"
else
    echo "✗ npm 未安装"
fi

# 检查后端关键包
echo ""
echo "=== 检查后端依赖 ==="
cd backend 2>/dev/null
python3 << 'PYTHON_EOF'
import sys
packages = {
    'fastapi': 'FastAPI',
    'uvicorn': 'Uvicorn',
    'fitz': 'PyMuPDF',
    'pdfplumber': 'pdfplumber',
    'camelot': 'camelot',
    'cv2': 'OpenCV',
    'yaml': 'PyYAML',
    'pandas': 'pandas',
    'numpy': 'numpy'
}

for module, name in packages.items():
    try:
        __import__(module)
        print(f"✓ {name}")
    except ImportError:
        print(f"✗ {name} - 未安装")

PYTHON_EOF

# 检查前端依赖
echo ""
echo "=== 检查前端依赖 ==="
cd ../frontend 2>/dev/null
if [ -d "node_modules" ]; then
    echo "✓ node_modules 目录存在"
    if [ -d "node_modules/react" ]; then
        echo "✓ React 已安装"
    else
        echo "✗ React 未安装"
    fi
    if [ -d "node_modules/vite" ]; then
        echo "✓ Vite 已安装"
    else
        echo "✗ Vite 未安装"
    fi
else
    echo "✗ node_modules 不存在 - 运行: cd frontend && npm install"
fi

echo ""
echo "=== 检查完成 ==="
EOF

chmod +x check_dependencies.sh
./check_dependencies.sh
```

## 🐛 常见问题

### 问题 1: 安装 OpenCV 失败
```bash
# 解决方案：先安装 opencv
brew install opencv
pip3 install opencv-python
```

### 问题 2: 安装 Camelot 失败
```bash
# 解决方案：需要系统依赖
brew install ghostscript imagemagick
pip3 install camelot-py[cv]
```

### 问题 3: npm 安装慢或失败
```bash
# 解决方案：使用国内镜像
npm config set registry https://registry.npmmirror.com

# 或者使用淘宝镜像
npm config set registry https://registry.npm.taobao.org
```

### 问题 4: Python 包安装权限问题
```bash
# 解决方案：使用用户安装
pip3 install --user -r backend/requirements.txt

# 或使用虚拟环境（推荐）
python3 -m venv venv
source venv/bin/activate
pip install -r backend/requirements.txt
```

### 问题 5: LaTeX 编译失败
```bash
# 确保安装了完整的 MacTeX
brew install --cask mactex

# 安装中文字体支持
brew install --cask mactex-no-gui
```

## 📝 快速安装脚本

如果您已安装 Homebrew 和 Xcode Command Line Tools，可以运行：

```bash
cat > quick_install.sh << 'EOF'
#!/bin/bash
set -e

echo "开始快速安装..."

# 安装 Python 依赖
echo "安装 Python 包..."
pip3 install --upgrade pip
pip3 install -r backend/requirements.txt

# 安装前端依赖
echo "安装前端依赖..."
cd frontend
npm install
cd ..

# 安装论文模板依赖
echo "安装论文模板依赖..."
pip3 install matplotlib plantuml requests

echo "✓ 安装完成！"
EOF

chmod +x quick_install.sh
./quick_install.sh
```
