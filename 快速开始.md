# 🚀 快速开始指南

## ⚠️ 检测到的系统状态

您的系统当前缺少必要的开发工具：

- ❌ **Xcode Command Line Tools** - 必须安装
- ❌ **Python 环境** - 依赖上面的工具
- ❌ **Node.js** - 前端需要

---

## 📋 第一步：安装 Xcode Command Line Tools

### 为什么需要这个？

Xcode Command Line Tools 是 macOS 上所有开发工具的基础，包括：
- Git
- Python
- GCC 编译器
- 其他开发工具

**如果没有这个，Python 和 Node.js 都无法正常工作！**

### 如何安装

在**终端**中运行以下命令：

```bash
xcode-select --install
```

### 会发生什么？

1. 会弹出一个对话框（如下图所示）
2. 点击 **"安装"** 按钮
3. 等待下载和安装（10-30 分钟，取决于网速）
4. 安装完成后点击 **"完成"**

![安装对话框](https://raw.githubusercontent.com/webpro/programming-setup/master/macos/images/command-line-tools-install-dialog.png)

### 验证安装

安装完成后，在终端运行：

```bash
xcode-select -p
```

应该输出类似：
```
/Library/Developer/CommandLineTools
```

4. ✅ 安装前端依赖（如果 Node.js 已安装）

## 📋 第三步：安装 Node.js（前端需要）

如果您需要运行前端应用：

### 方法 1：使用 Homebrew（推荐）

```bash
# 先安装 Homebrew（如果没有）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Node.js
brew install node

# 验证
node --version
npm --version
```

### 方法 2：官方安装包

访问 https://nodejs.org/ 下载并安装 LTS 版本

---

## ✅ 验证安装

安装完成后运行检查脚本：

```bash
./check_dependencies.sh
```

所有项都应该显示 ✓ 绿色标记

## 🚀 启动项目

### 启动后端（终端 1）

```bash
source venv/bin/activate
cd backend
python3 main.py
```

后端将在 http://localhost:8000 启动

### 启动前端（终端 2，需要 Node.js）

```bash
cd frontend
npm run dev
```

前端将在 http://localhost:5173 启动
