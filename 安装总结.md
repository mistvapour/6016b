## 💡 重要提示

1. **必须先安装 Xcode Command Line Tools**
   - macOS 开发必需
   - 包含编译工具
   - 安装时间较长

2. **建议使用虚拟环境**
   - 避免依赖冲突
   - 安装脚本会自动创建
   - 路径: `venv/`

3. **分步安装更稳定**
   - 如果一键安装失败
   - 按照 INSTALL.md 逐步安装
   - 每步验证后再继续

4. **检查依赖状态**
   - 运行 `./check_dependencies.sh`
   - 查看是否有 ✗ 标记
   - 根据提示补装缺失依赖
