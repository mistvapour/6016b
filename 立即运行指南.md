# 🚀 立即运行指南

## ⚠️ 重要：需要用户手动操作

由于需要系统权限和用户交互，某些步骤需要您在**终端手动运行**。

---

## 🎯 完整安装步骤

### 步骤 1️⃣: 安装 Xcode Command Line Tools

**请在终端运行以下命令：**

```bash
xcode-select --install
```

**会出现什么：**
- 弹出一个对话框
- 点击 **"安装"** 按钮
- 等待 10-30 分钟
- 完成后点击 **"完成"**

**验证安装成功：**
```bash
xcode-select -p
```
应该显示：`/Library/Developer/CommandLineTools`

---

### 步骤 2️⃣: 安装项目依赖

Xcode 工具安装完成后，**在终端运行：**

```bash
# 进入项目目录
cd /Users/feilubi/Documents/GitHub/6016b

# 运行自动安装脚本
./install.sh
```

**这个脚本会：**
1. 检查系统环境
2. 创建 Python 虚拟环境
3. 安装所有后端依赖
4. 安装前端依赖（如果有 Node.js）
5. 验证安装

---

### 步骤 3️⃣: 验证安装

**运行检查脚本：**

```bash
./check_dependencies.sh
```

**应该看到所有 ✓ 绿色标记！**

---

### 步骤 4️⃣: 启动项目

**启动后端（终端 1）：**

```bash
source venv/bin/activate
cd backend
python3 main.py
```

后端将在 http://localhost:8000 启动

**启动前端（终端 2，需要 Node.js）：**

```bash
cd frontend
npm run dev
```

前端将在 http://localhost:5173 启动

---

## 🔄 如果 install.sh 不工作

### 手动安装后端依赖

```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate

# 安装依赖
pip install -r backend/requirements.txt
```

### 手动安装前端依赖

```bash
cd frontend
npm install
cd ..
```

## 🐛 常见问题

### Q1: 提示 "command not found"

**原因**: Xcode 工具未安装

**解决**: 运行 `xcode-select --install`

### Q2: pip 安装失败

**解决**: 
```bash
# 升级 pip
python3 -m pip install --upgrade pip

# 使用国内镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r backend/requirements.txt
```

### Q3: npm 安装慢或失败

**解决**:
```bash
# 使用国内镜像
npm config set registry https://registry.npmmirror.com

# 重新安装
cd frontend
rm -rf node_modules package-lock.json
npm install
```
