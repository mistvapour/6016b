# 🚀 后端服务启动说明（使用 localhost:8000）

## ✅ 正确的启动命令

在 Git Bash 中运行：

```bash
cd backend
source .venv/Scripts/activate
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

## 📍 服务地址

- **后端 API**: http://localhost:8000
- **API 文档**: http://localhost:8000/docs  
- **健康检查**: http://localhost:8000/api/health

## ⚙️ 启动参数说明

- `--host 127.0.0.1`: 绑定到 localhost，只允许本地访问（更安全）
- `--port 8000`: 使用 8000 端口
- `--reload`: 开发模式，代码修改后自动重启

## 🔄 重启后端

使用批处理文件：
- 双击运行 `backend/重启后端.bat`

或手动重启：
```bash
# 停止现有服务（按 Ctrl+C）
# 然后重新运行启动命令
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

## ⚠️ 注意事项

- 使用 `127.0.0.1` 或 `localhost` 只允许本地访问
- 如果需要从其他设备访问，请使用 `--host 0.0.0.0`
- 不要直接运行 `python main.py`，会导致 Segmentation fault
