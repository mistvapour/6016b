# 如何查看导出的样例结果

运行批量测试后，样例结果有三种查看方式。

---

## 1. 控制台输出（即时查看）

运行命令后，终端会直接打印：

- **量化评估结果**：物理闭合合格率、逻辑合规率
- **样例结果文件路径**：如 `.../logs/bsc_sample_instances_20250129_123456.json`
- **详细日志文件路径**：如 `.../logs/bsc_binary_instance_20250129_123456.log`

示例：

```
========== 量化评估结果 ==========
  物理闭合合格率: 100/100 = 100.0%
  逻辑合规率:     100/100 = 100.0%
  详细日志文件:   C:\...\backend\logs\bsc_binary_instance_20250129_123456.log
  样例结果文件:   C:\...\backend\logs\bsc_sample_instances_20250129_123456.json
==================================
```

---

## 2. 样例结果 JSON 文件（字段值 + 二进制 + 十六进制）

- **位置**：默认在 `backend/logs/` 下，文件名形如 `bsc_sample_instances_YYYYMMDD_HHMMSS.json`
- **内容**：前 N 条生成实例的 `field_values`、`binary`、`hex`、`bit_length`（N 由 `--export-samples` 控制，默认 5）

**如何查看**：

- 用任意文本编辑器或 VS Code 打开该 `.json` 文件
- 或用命令行：`type logs\bsc_sample_instances_*.json`（Windows）、`cat logs/bsc_sample_instances_*.json`（Linux/Mac）

**JSON 结构示例**：

```json
{
  "total_bits": 24,
  "count": 5,
  "samples": [
    {
      "index": 1,
      "field_values": { "Field_A": 42, "Field_B": 180, "Field_C": 10 },
      "binary": "001010101011010000001010",
      "hex": "2AB40A",
      "bit_length": 24
    }
  ]
}
```

- **field_values**：各字段的整型编码值（用于按位拼接）
- **binary**：整条报文的二进制串（0/1）
- **hex**：整条报文的十六进制报文
- **bit_length**：当前实例的位长（应与 `total_bits` 一致）

---

## 3. 详细日志文件（论文截取用）

- **位置**：默认在 `backend/logs/` 下，文件名形如 `bsc_binary_instance_YYYYMMDD_HHMMSS.log`
- **内容**：每条实例的约束决策（如「条件成立→生成」「条件不成立→填 0」）、物理/逻辑校验记录、统计汇总等

**如何查看**：

- 用记事本、VS Code 等打开 `.log` 文件
- 搜索标签可快速定位：
  - `[BSC-GEN][Constraint]`：约束是否满足、是否生成该字段
  - `[BSC-GEN][PhysicalClosure]`：位长是否合格
  - `[BSC-GEN][LogicCompliance]`：逻辑是否合规
  - `[BSC-GEN][Stats]`：批量统计

---

## 常用命令

```bash
cd backend

# 默认：生成 100 条，导出前 5 条样例到 logs/
python test_generate_binary_instance.py

# 导出前 10 条样例
python test_generate_binary_instance.py --export-samples 10

# 不导出样例（仅统计 + 日志）
python test_generate_binary_instance.py --export-samples 0

# 指定输出目录
python test_generate_binary_instance.py --out-dir output
```

导出后，用资源管理器或 `dir logs` 查看 `logs/` 下最新生成的 `bsc_sample_instances_*.json` 和 `bsc_binary_instance_*.log` 即可。
