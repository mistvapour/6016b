# 🔧 后端服务启动说明

## ⚠️ 重要提示

**不要直接运行 `python main.py`**，这会导致 Segmentation fault 错误！

## ✅ 正确的启动方式

### 方法 1: 使用 uvicorn 命令（推荐）

```bash
cd backend
source .venv/Scripts/activate
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 方法 2: 使用批处理文件（Windows）

双击运行 `backend/启动后端.bat` 文件

### 方法 3: 使用 PowerShell/CMD

```cmd
cd backend
.venv\Scripts\activate.bat
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

## 🔍 启动参数说明

- `--host 0.0.0.0`: 允许从任何网络接口访问（不仅仅是 localhost）
- `--port 8000`: 指定端口号
- `--reload`: 开启自动重载（开发模式，代码修改后自动重启）

## 📋 验证服务

启动成功后，访问：
- **健康检查**: http://localhost:8000/api/health
- **API 文档**: http://localhost:8000/docs

## ❌ 常见错误

### Segmentation fault

如果遇到 Segmentation fault，请：
1. 确保使用 `python -m uvicorn` 而不是直接运行 `python main.py`
2. 检查虚拟环境是否正确激活
3. 确保所有依赖都已安装：`pip install -r requirements.txt`

## 🛑 停止服务

按 `Ctrl+C` 即可停止服务
