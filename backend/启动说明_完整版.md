# 🚀 后端启动完整指南

## ❌ 问题：连接被拒绝

**错误信息**：`127.0.0.1 拒绝了我们的连接请求`

**原因**：后端服务没有运行

## ✅ 解决方案

### 方法 1: 使用批处理脚本（最简单）

**双击运行以下任一文件：**

1. `backend/快速启动后端.bat` - 新增，包含完整检查
2. `backend/启动后端.bat` - 原有启动脚本

### 方法 2: 在终端中手动启动

#### Git Bash 终端：

```bash
cd backend
source .venv/Scripts/activate
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

#### PowerShell/CMD 终端：

```cmd
cd backend
.venv\Scripts\activate.bat
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

## ⚠️ 重要提示

1. **后端服务必须在独立的终端窗口运行**
2. **不要关闭运行后端的终端窗口**
3. **使用 `Ctrl+C` 停止服务**
4. **如果修改了代码，服务会自动重载（reload 模式）**

## 🔧 故障排除

### 如果启动失败：

1. **检查虚拟环境**：
   ```bash
   ls -la .venv/Scripts
   ```

2. **检查依赖**：
   ```bash
   pip list | grep -E "fastapi|uvicorn"
   ```

3. **重新安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
