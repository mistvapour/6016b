### 如需启动前端

#### 选项 1：安装 Node.js（推荐）

```bash
# 使用 Homebrew 安装
brew install node

# 验证安装
node --version
npm --version
```

#### 选项 2：下载官方安装包

访问 https://nodejs.org/ 下载 LTS 版本

### 安装后启动前端

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

前端将在 http://localhost:5173 启动

### 2. 使用命令行工具

后端提供的所有功能都可以通过 API 调用：

```bash
# PDF 处理
curl -X POST http://localhost:8000/api/pdf/process \
  -F "file=@test_sample/link16-import.pdf" \
  -F "standard=MIL-STD-6016" \
  -F "edition=B"

# 查看统计
curl http://localhost:8000/api/statistics
```

### 3. 运行演示脚本

```bash
# 激活虚拟环境
source venv/bin/activate

python3 demo_vector_rag.py
```
